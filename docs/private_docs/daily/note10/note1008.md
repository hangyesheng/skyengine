1. 支持离散事件随机生成,包括伯努利、泊松、几何分布这三种离散型分布。
2. 支持设置时间quant,默认为0.1s。
3. 支持train。

三种模式都能看到自己的感受野,区别在于MAPF同时看到全局;
而POMAPF和DEFAULT一般来说应该没啥区别

默认模式 (3, obs_radius*2+1, obs_radius*2+1)的数组
- 第一层 (索引0) : 障碍物层 - 表示智能体视野范围内的障碍物位置
- 第二层 (索引1) : 智能体层 - 表示智能体视野范围内的其他智能体位置
- 第三层 (索引2) : 目标层 - 表示智能体的目标位置

POMAPF模式提供部分可观察的多智能体路径规划观察空间，返回一个字典，包含：
- obstacles : 智能体视野范围内的障碍物位置 (形状为 (obs_radius*2+1, obs_radius*2+1) )
- agents : 智能体视野范围内的其他智能体位置 (形状为 (obs_radius*2+1, obs_radius*2+1) )
- xy : 智能体当前的相对坐标 (形状为 (2,) )
- target_xy : 智能体目标的相对坐标 (形状为 (2,) )

MAPF模式提供完全可观察的多智能体路径规划观察空间，在POMAPF的基础上增加了全局信息：
- 包含POMAPF模式的所有信息
- global_obstacles : 全局障碍物位置
- global_xy : 智能体在全局网格中的坐标
- global_target_xy : 智能体目标在全局网格中的坐标


POMAPF模式的设计更加结构化和明确，便于算法直接访问特定类型的信息，而默认模式则更适合直接输入到神经网络等模型中进行处理。
