




JobSolver 负责“谁干什么，在哪干”；

RouteSolver 负责“怎么过去，什么时候到”；

两者接口统一，可独立测试或级联执行；

RL 代理可在 job 层学习调度策略，而 route 层保持物理仿真。

| 类别   | 指标                             | 含义           |
| ---- | ------------------------------ | ------------ |
| 成功率  | success_rate                   | 任务是否完成       |
| 路径质量 | avg_path_length / total_length | 路径长度         |
| 效率   | avg_completion_steps           | 平均到达步数       |
| 拥堵性  | conflict_count / wait_steps    | 冲突或等待次数      |
| 平衡性  | path_length_std                | 不同 AGV 负载均衡度 |
